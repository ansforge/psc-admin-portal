Exécution complète du processus

Souhaitez-vous supprimer l'extraction RASS en Anomalie avant de charger le RASS :
  1. Charger le RASS
  2. Traiter le fichier RASS
  3. Lever des alertes (éventuellement)
  4. Enregistrer les PS en base
  5. Générer le fichier sécurisé
  6. Envoyer les PS à l'Imprimerie Nationale
